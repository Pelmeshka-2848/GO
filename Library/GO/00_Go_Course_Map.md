# Go Backend Developer Course

## Цель курса

Освоить backend-разработку на Go: от базового синтаксиса до REST API, PostgreSQL, конкурентности, архитектуры и микросервисов.

## Главный результат

К концу курса должны быть:

- теоретическая база по Go;
- набор лабораторных работ;
- набор мини-кейсов;
- база ошибок и решений;
- база кодовых рецептов;
- финальный backend-проект.

## Правило чистого графа Obsidian

- Практика может ссылаться на теорию.
- Практика может ссылаться на ошибки и рецепты.
- Теория может ссылаться на другую теорию.
- Теория не должна массово ссылаться на каждую лабораторную.
- Шаблоны не должны связываться с рабочими заметками.
