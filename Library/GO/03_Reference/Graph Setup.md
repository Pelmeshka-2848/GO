# Graph Setup

Чтобы граф выглядел разделенным:

## Вариант 1. Через фильтр

В Graph View включить фильтр:

```text
path:01_Theory OR path:02_Practice
```

## Вариант 2. Смотреть только теорию

```text
path:01_Theory
```

## Вариант 3. Смотреть только практику

```text
path:02_Practice
```

## Вариант 4. Скрыть шаблоны и справку

```text
-path:04_Templates -path:03_Reference
```

## Рекомендуемая логика связей

- Теория может ссылаться на другую теорию.
- Практика может ссылаться на теорию.
- Теория не должна массово ссылаться на каждую лабораторную.
- Шаблоны лучше не связывать с рабочими заметками.
