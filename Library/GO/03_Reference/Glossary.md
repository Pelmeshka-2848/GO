# Glossary

| Термин | Объяснение | Раздел |
|---|---|---|
| handler | Функция или объект, который принимает HTTP-запрос | backend |
| repository | Слой работы с хранилищем данных | architecture |
| DTO | Объект передачи данных между слоями или через API | architecture |
| middleware | Промежуточный обработчик запроса | backend |
