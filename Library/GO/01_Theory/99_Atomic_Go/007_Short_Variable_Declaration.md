# Short Variable Declaration

## Суть

Оператор `:=` создаёт новую переменную и сразу присваивает ей значение.

```go
name := "Гоша"
```

Go сам выводит тип переменной из значения. В примере `name` получает тип `string`.

## Когда использовать

`:=` используется внутри функций.

```go
func main() {
	count := 10
}
```

На уровне пакета нужно использовать `var`.

```go
var count = 10
```
