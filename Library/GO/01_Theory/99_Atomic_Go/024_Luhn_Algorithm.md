# Luhn Algorithm

## Суть

Алгоритм Луна — алгоритм проверки контрольной суммы номера карты.

## Логика

1. Идём по цифрам справа налево.
2. Каждую вторую цифру удваиваем.
3. Если результат удвоения больше 9, вычитаем 9.
4. Складываем все получившиеся цифры.
5. Если сумма делится на 10, номер проходит проверку.

## Финальная проверка

```go
return sum%10 == 0
```

## Важное ограничение

Алгоритм Луна проверяет контрольную сумму, но не подтверждает, что карта реально существует, активна или принадлежит конкретному банку.
