# Reverse For Loop

## Суть

Для обхода строки или slice справа налево используется индексный цикл.

```go
for i := len(cardNumber) - 1; i >= 0; i-- {
	ch := cardNumber[i]
}
```

## Где используется

Такой цикл нужен в алгоритме Луна, потому что обработка начинается с конца номера карты.

## Gotcha

Если строка пустая, `len(cardNumber)-1` даёт `-1`. Поэтому пустой ввод лучше отсеять заранее.

```go
if cardNumber == "" {
	return false
}
```
