# Standard Input

## Суть

`stdin` — стандартный поток ввода. Обычно это данные, которые пользователь вводит в терминале или передаёт программе через pipe.

В Go доступ к стандартному вводу даёт `os.Stdin`.

```go
scanner := bufio.NewScanner(os.Stdin)
```

## Пример запуска

```bash
echo "Гоша" | go run main.go
```

Здесь строка `Гоша` передаётся программе через стандартный ввод.
