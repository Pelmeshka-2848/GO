# Exported And Unexported Names

## Суть

В Go регистр первой буквы имени определяет видимость между пакетами.

## Экспортируемые имена

```go
type Bank struct {
	Name   string
	Prefix string
}
```

`Bank`, `Name`, `Prefix` начинаются с большой буквы, поэтому доступны из других пакетов.

## Неэкспортируемые имена

```go
type bank struct {
	name   string
	prefix string
}
```

Такие имена доступны только внутри текущего пакета.

## Практическое правило

Если тип или поле должны быть доступны извне, называй их с большой буквы.
