# ASCII Digit Conversion

## Суть

Чтобы превратить ASCII-цифру в число, используется выражение:

```go
digit := int(ch - '0')
```

## Почему работает

Символы `'0'` — `'9'` идут подряд в ASCII.

```text
'0' -> 48
'1' -> 49
...
'9' -> 57
```

Поэтому:

```go
'7' - '0' == 7
```

## Обязательная проверка

Перед преобразованием нужно проверить диапазон:

```go
if ch < '0' || ch > '9' {
	return false
}
```
