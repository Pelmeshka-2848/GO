# Time Format Layout

## Суть

В Go форматирование времени устроено через опорное время, а не через `%H:%M` или `HH:mm`.

Опорное время:

```text
Mon Jan 2 15:04:05 MST 2006
```

## Формат часы:минуты

```go
formatted := now.Format("15:04")
```

## Шпаргалка

| Часть layout | Значение |
|---|---|
| `15` | часы в 24-часовом формате |
| `04` | минуты |
| `05` | секунды |
| `02` | день |
| `01` | месяц |
| `2006` | год |

## Gotcha

Нельзя писать:

```go
now.Format("%H:%M")
```

В Go это не сработает как в Python.
