# nil

## Суть

`nil` — нулевое значение для некоторых ссылочных типов: указателей, slice, map, channel, function и interface.

## Пример с указателем

```go
func DetectBank(cardNumber string, banks []Bank) *Bank {
	return nil
}
```

Здесь `nil` означает, что банк не найден.

## Проверка результата

```go
bank := DetectBank("4000123456789017", banks)

if bank != nil {
	fmt.Println(bank.Name)
}
```
