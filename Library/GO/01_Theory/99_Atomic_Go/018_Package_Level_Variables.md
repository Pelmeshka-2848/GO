# Package Level Variables

## Суть

Переменная может быть объявлена на уровне функции или на уровне пакета.

## Внутри функции

```go
func main() {
	banks := []Bank{}
}
```

Такая переменная доступна только внутри функции.

## На уровне пакета

```go
var banks = []Bank{}
```

Такая переменная доступна всем функциям текущего пакета.

## Когда использовать

Список справочных данных, который нужен нескольким функциям, удобно держать на уровне пакета.
