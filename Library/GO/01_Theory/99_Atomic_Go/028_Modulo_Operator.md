# Modulo Operator

## Суть

Оператор `%` возвращает остаток от деления.

```go
70 % 10 == 0
```

## Применение в алгоритме Луна

Если сумма делится на 10 без остатка, номер проходит проверку контрольной суммы.

```go
return sum%10 == 0
```
