# String Indexing Bytes

## Суть

В Go обращение к строке по индексу возвращает `byte`, а не `rune`.

```go
ch := cardNumber[i]
```

Для ASCII-цифр это нормально, потому что каждая цифра занимает один байт.

## Gotcha

```go
len("Гоша")
```

вернёт количество байт, а не количество букв.

Для Unicode-символов индексный обход строки не равен обходу по символам.

## Когда индексный обход подходит

Для номера карты индексный обход безопасен, если строка должна состоять только из ASCII-цифр `'0'` — `'9'`.
