# fmt.Printf

## Суть

`fmt.Printf` выводит строку по формату и позволяет точно управлять расположением значений.

```go
fmt.Printf("Привет, %s! Сейчас %s\n", name, formatted)
```

## Частые спецификаторы

| Спецификатор | Значение |
|---|---|
| `%s` | строка |
| `%d` | целое число |
| `%v` | значение в стандартном формате |
| `%T` | тип значения |

## Отличие от `fmt.Println`

`fmt.Println` просто выводит аргументы через пробел.

`fmt.Printf` позволяет управлять пунктуацией, пробелами и переносами строк.
