# fmt Package

## Суть

`fmt` — стандартный пакет Go для форматированного ввода и вывода.

## Частые функции

| Функция | Назначение |
|---|---|
| `fmt.Println` | выводит аргументы через пробел и добавляет перевод строки |
| `fmt.Printf` | выводит строку по формату |
| `fmt.Sprintf` | возвращает строку по формату без вывода в консоль |

## `fmt.Println`

```go
fmt.Println("Привет,", name)
```

Особенности:

- автоматически добавляет пробелы между аргументами;
- автоматически добавляет `\n` в конце;
- подходит для простого вывода.
