# Code Generation

## Что это

Code generation — автоматическое создание исходного кода на основе схем, интерфейсов, спецификаций или шаблонов.

## Зачем нужно

Генерация уменьшает ручную рутину и помогает держать код синхронизированным с контрактами.

## Где встречается

- mocks;
- OpenAPI clients/servers;
- protobuf/gRPC;
- SQL query wrappers;
- stringer-like tools;
- dependency injection tooling.

## Правила

- Генерируемый код должен быть воспроизводимым.
- Команда генерации должна быть зафиксирована.
- Не редактируй generated-файлы вручную.
- Generated-код должен быть отделен от ручного кода.
- Не применяй generation там, где обычный код проще.

## Типичные ошибки

- забыть закоммитить обновленный generated-код, если проект так договорился;
- изменить generated-файл руками;
- не указать версию generator-а;
- усложнить проект генерацией раньше времени.
