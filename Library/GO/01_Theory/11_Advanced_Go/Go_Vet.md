# Go Vet

## Что это

`go vet` — стандартный анализатор подозрительных конструкций в Go-коде.

## Зачем нужно

Он находит проблемы, которые компилятор может пропустить: неправильные форматные строки, подозрительные копирования, unreachable patterns.

## Запуск

```bash
go vet ./...
```

## Правила

- Запускай `go vet` перед важными изменениями.
- Не считай `go vet` заменой тестам.
- Разбирай предупреждение до подавления.
- В CI `vet` полезен рядом с `go test`.

## Типичные ошибки

- игнорировать warning;
- запускать только на одном пакете;
- ожидать, что `vet` найдет все bugs;
- путать `vet` и `staticcheck`.
