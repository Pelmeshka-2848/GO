# Slices

## Что это

Slice — это динамическое представление последовательности элементов одного типа.

```go
numbers := []int{1, 2, 3}
```

В отличие от массива, длина slice может меняться через `append`.

## Зачем нужно

Слайсы используются почти везде, где нужен список: набор пользователей, строки файла, результаты запроса, очередь задач.

## Длина и вместимость

У slice есть:

- `len` — сколько элементов сейчас доступно;
- `cap` — сколько элементов может поместиться в текущий underlying array до перевыделения.

```go
items := make([]string, 0, 10)
```

Так создается пустой slice с заранее выделенной вместимостью 10.

## Добавление элементов

```go
items = append(items, "first")
```

Важно: результат `append` нужно присвоить обратно, потому что он может вернуть новый slice.

## Ключевые правила

- Используй slice для списков переменной длины.
- Не забывай присваивать результат `append`.
- При заранее известном примерном размере используй `make` с capacity.
- Помни, что несколько slice могут ссылаться на один underlying array.

## Частые ошибки

- путать array и slice;
- ожидать, что `append` изменит slice без присваивания;
- брать адрес переменной цикла `range`;
- случайно изменить общий underlying array через другой slice.
