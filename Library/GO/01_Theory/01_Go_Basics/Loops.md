# Loops

## Что это

Цикл выполняет один и тот же блок кода несколько раз. В Go есть один основной оператор цикла — `for`.

```go
for i := 0; i < 3; i++ {
    fmt.Println(i)
}
```

## Варианты `for`

Цикл с условием:

```go
for count < 10 {
    count++
}
```

Бесконечный цикл:

```go
for {
    // выход обычно через break или return
}
```

Обход коллекции:

```go
for index, value := range items {
    fmt.Println(index, value)
}
```

## Зачем нужно

Циклы используются для:

- обхода slice и map;
- повторной обработки ввода;
- чтения строк;
- обработки очереди задач;
- retry-логики с ограничением.

## Ключевые правила

- Для обхода slice и map чаще используй `range`.
- Если индекс не нужен, замени его на `_`.
- Следи за условием выхода из бесконечного цикла.
- Не меняй коллекцию во время обхода, если не понимаешь последствия.

## Частые ошибки

- взять адрес range-переменной;
- забыть обновить счетчик в цикле с условием;
- получить бесконечный цикл;
- ожидать стабильный порядок обхода map.
