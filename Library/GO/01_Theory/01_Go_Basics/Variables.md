# Variables

## Что это

Переменная в Go — это именованное место для значения определенного типа. Тип может быть указан явно или выведен компилятором из присваиваемого значения.

## Зачем нужно

Переменные нужны, чтобы хранить промежуточные данные: ввод пользователя, результат вычисления, конфигурацию, состояние цикла, ошибку или объект предметной области.

## Основные способы объявления

```go
var name string
var age int = 30
city := "Vladivostok"
```

`var` можно использовать внутри и вне функций. Короткая форма `:=` работает только внутри функций.

## Нулевые значения

Если переменная объявлена без значения, Go присваивает ей нулевое значение:

| Тип | Нулевое значение |
|---|---|
| `int` | `0` |
| `float64` | `0` |
| `bool` | `false` |
| `string` | `""` |
| pointer, slice, map, interface | `nil` |

## Ключевые правила

- Неиспользуемая локальная переменная вызывает ошибку компиляции.
- Имя переменной должно объяснять роль значения, а не только его тип.
- Используй `:=` для коротких локальных значений.
- Используй `var`, когда нужно нулевое значение или область видимости вне функции.
- Не переиспользуй одну переменную для разных смыслов.

## Частые ошибки

- объявить переменную и не использовать ее;
- случайно создать новую переменную через `:=`, когда хотел изменить старую;
- назвать переменную слишком общо: `data`, `value`, `tmp`;
- хранить ошибку в переменной `err`, но не проверять ее.
