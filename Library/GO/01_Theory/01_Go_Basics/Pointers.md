# Pointers

## Что это

Указатель хранит адрес значения в памяти.

```go
value := 10
ptr := &value
```

`&` берет адрес значения, `*` получает значение по адресу.

```go
fmt.Println(*ptr)
```

## Зачем нужно

Указатели используются, когда нужно:

- изменить значение внутри функции;
- не копировать большую структуру;
- показать отсутствие значения через `nil`;
- работать с методами, которые меняют состояние объекта.

## Pointer receiver

Метод с pointer receiver может менять структуру:

```go
func (u *User) Rename(name string) {
    u.Name = name
}
```

Метод со value receiver работает с копией значения.

## Ключевые правила

- Не используй указатель только “потому что так быстрее”.
- Для маленьких неизменяемых значений копирование часто проще.
- Проверяй `nil`, если указатель может отсутствовать.
- Если метод меняет структуру, чаще всего нужен pointer receiver.
- Будь осторожен с адресом переменной цикла.

## Частые ошибки

- dereference `nil` pointer;
- ожидать, что функция изменит значение, переданное по значению;
- возвращать указатели на временные значения без понимания escape analysis;
- брать адрес range-переменной и получать повторяющиеся значения.
