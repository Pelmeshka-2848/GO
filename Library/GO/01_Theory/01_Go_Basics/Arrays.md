# Arrays

## Что это

Array — это последовательность элементов одного типа с фиксированной длиной.

```go
var numbers [3]int
names := [2]string{"Anna", "Ivan"}
```

Длина массива является частью его типа. `[3]int` и `[4]int` — разные типы.

## Зачем нужно

В обычной Go-разработке чаще используются slice, но массивы важны для понимания:

- как устроены слайсы;
- что значит фиксированная длина;
- почему копирование массива может быть дорогим;
- как работать с небольшими фиксированными наборами.

## Нулевые значения

Если массив объявлен без значений, каждый элемент получает нулевое значение своего типа.

```go
var flags [3]bool
// [false false false]
```

## Ключевые правила

- Используй array, когда размер действительно фиксирован и важен.
- Для списков переменной длины используй slice.
- Помни, что массив при присваивании копируется целиком.
- Индекс начинается с нуля.

## Частые ошибки

- пытаться добавить элемент в array через `append`;
- путать `[3]int` и `[]int`;
- забывать, что длина входит в тип массива;
- передавать большой массив по значению.
