# fmt

## Кратко

`fmt` — стандартный пакет для форматированного ввода и вывода: печать в консоль, сборка строк, форматирование значений.

## Зачем нужно

- выводить учебные результаты в консоль;
- форматировать ошибки и строки;
- отлаживать значения на ранних этапах;
- готовить человекочитаемый текст.

## Основные функции

- `fmt.Println` — печать с переводом строки;
- `fmt.Printf` — печать по формату;
- `fmt.Sprintf` — вернуть отформатированную строку;
- `fmt.Errorf` — создать ошибку с форматированием.

## Частые спецификаторы

| Формат | Смысл |
|---|---|
| `%s` | строка |
| `%d` | целое число |
| `%v` | значение в стандартном виде |
| `%+v` | значение с дополнительными деталями |
| `%T` | тип значения |
| `%w` | оборачивание ошибки в `fmt.Errorf` |

## Типичные ошибки

| Ошибка | Почему возникает | Как исправить |
|---|---|---|
| Неверный формат | `%d` применили к строке | Подобрать спецификатор под тип |
| Использовать `Println` для форматной строки | `Println("x=%d", x)` не форматирует | Использовать `Printf` |
| Потерять исходную ошибку | `fmt.Errorf("failed: %v", err)` | Для wrapping использовать `%w` |
