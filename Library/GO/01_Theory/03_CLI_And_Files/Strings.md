# Strings

## Что это

`string` в Go — это неизменяемая последовательность байтов. Обычно строка содержит текст в UTF-8, но технически это именно bytes.

## Зачем нужно

Строки используются для ввода пользователя, путей файлов, HTTP-тел, JSON, логов, SQL-запросов и текстовой обработки.

## Важная разница

Индексация строки возвращает byte, а не rune:

```go
s := "Go"
first := s[0]
```

Для Unicode-символов нужно понимать разницу между byte и rune.

## Полезные операции

- сравнение строк;
- проверка префикса и суффикса;
- разбиение строки;
- объединение;
- обрезка пробелов;
- поиск подстроки;
- преобразование регистра.

Часто используется пакет `strings`.

## Ключевые правила

- Не меняй строку “на месте”: строки неизменяемы.
- Для большого количества склеек используй `strings.Builder`.
- Для Unicode-обхода используй `range`, а не индексирование по byte.
- Для пользовательского ввода часто нужен `strings.TrimSpace`.

## Частые ошибки

- считать `len(s)` количеством символов, хотя это количество байтов;
- индексировать Unicode-строку и получать byte;
- склеивать много строк через `+` в цикле;
- не обрезать перевод строки после ввода.
