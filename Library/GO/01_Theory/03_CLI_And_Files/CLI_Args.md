# CLI Args

## Что это

CLI arguments — это значения, которые пользователь передает программе при запуске из терминала.

```bash
go run . add 2 3
```

В Go сырые аргументы доступны через `os.Args`.

## Как это выглядит

`os.Args` — это slice строк:

- `os.Args[0]` — имя запущенной программы;
- `os.Args[1:]` — пользовательские аргументы.

## Зачем нужно

Аргументы командной строки используются для CLI-утилит: калькулятор, конвертер файлов, генератор отчетов, миграции, dev-команды.

## Ключевые правила

- Проверяй количество аргументов перед обращением по индексу.
- Помни, что все аргументы приходят строками.
- Для чисел нужно явное преобразование.
- Для сложных CLI лучше использовать пакет `flag` или CLI-библиотеку.
- Пользователь должен получать понятную подсказку при неверном запуске.

## Частые ошибки

- обратиться к `os.Args[1]`, когда аргументов нет;
- забыть проверить ошибку преобразования строки в число;
- смешать позиционные аргументы и флаги без понятного формата;
- не описать пример запуска.
