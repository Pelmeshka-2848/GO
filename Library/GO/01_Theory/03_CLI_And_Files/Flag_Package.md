# Flag Package

## Что это

`flag` — стандартный пакет Go для разбора CLI-флагов.

```bash
go run . -input notes.md -output notes.html
```

## Зачем нужно

Флаги делают CLI-команды понятнее, когда параметров больше одного или порядок аргументов не должен иметь значения.

## Типовые флаги

Пакет `flag` поддерживает строки, числа, boolean и другие базовые типы через функции вроде:

- `flag.String`;
- `flag.Int`;
- `flag.Bool`;
- `flag.Duration`.

После объявления флагов нужно вызвать разбор:

```go
flag.Parse()
```

## Ключевые правила

- У каждого флага должно быть понятное имя.
- Добавляй help-текст, чтобы `-h` был полезным.
- Не смешивай много позиционных аргументов и флагов без необходимости.
- Значение по умолчанию должно быть безопасным.
- После `flag.Parse()` можно читать оставшиеся позиционные аргументы через `flag.Args()`.

## Частые ошибки

- забыть вызвать `flag.Parse()`;
- использовать флаг без описания;
- выбирать опасное значение по умолчанию;
- ожидать, что `flag` сам поддерживает сложные subcommands.
