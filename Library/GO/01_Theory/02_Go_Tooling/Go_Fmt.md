# Go Fmt

## Что это

`go fmt` форматирует Go-код по стандартному стилю языка.

```bash
go fmt ./...
```

## Зачем нужно

В Go стиль форматирования стандартизирован. Это убирает споры о пробелах, скобках и выравнивании.

## Основные команды

```bash
go fmt .
go fmt ./...
gofmt -w main.go
```

`go fmt` работает на уровне пакетов. `gofmt` может форматировать конкретные файлы.

## Ключевые правила

- Форматируй код перед коммитом.
- Для всего модуля используй `go fmt ./...`.
- Не форматируй Go-код вручную, если инструмент делает это автоматически.
- IDE обычно может запускать форматирование при сохранении.

## Частые ошибки

- спорить со стандартным форматированием;
- забыть форматирование после массовых правок;
- запускать `go fmt` не из нужной папки;
- путать `go fmt` и `go vet`.
