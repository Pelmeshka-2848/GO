# Go Vet

## Что это

`go vet` ищет подозрительные конструкции, которые компилируются, но часто являются ошибками.

```bash
go vet ./...
```

## Зачем нужно

Компилятор проверяет синтаксис и типы. `go vet` дополнительно ловит распространенные проблемы: неправильное форматирование аргументов, подозрительные копирования lock-ов и другие ошибки.

## Ключевые правила

- Запускай `go vet ./...` как базовую проверку качества.
- Не воспринимай `go vet` как замену тестам.
- Если `vet` ругается, сначала разберись с причиной, а не подавляй предупреждение.
- В CI `go vet` часто идет рядом с `go test`.

## Частые ошибки

- игнорировать предупреждения;
- считать, что если `go build` прошел, то код точно корректен;
- запускать `vet` только на одном пакете;
- путать `go vet` со сторонними линтерами.
