# Ошибка декодирования JSON

## Контекст

Возникает в HTTP API или CLI-утилите, когда программа пытается прочитать JSON, но входные данные не соответствуют ожидаемому формату.

## Текст ошибки

```text
invalid character
unexpected EOF
json: cannot unmarshal
```

## Причина

JSON может быть синтаксически неправильным, тело запроса может быть пустым, тип поля в JSON может не совпадать с типом поля в struct.

## Решение

- Проверить тело запроса или содержимое файла.
- Убедиться, что JSON валиден.
- Сравнить поля JSON с Go-структурой.
- Вернуть понятную ошибку клиенту, если JSON невалидный.

## Как не повторить

- Проверять JSON отдельным минимальным примером.
- Добавлять валидацию после декодирования.
- Не смешивать ошибку формата JSON с ошибкой бизнес-логики.
