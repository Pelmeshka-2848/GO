# Git Problems

## Симптомы

```text
not a git repository
Your branch is behind
Please commit your changes or stash them before you merge
merge conflict
```

## Где возникло

- проект: vault или учебный Go-проект;
- команда: `git status`, `git pull`, `git add`, `git commit`;
- окружение: локальный репозиторий, удаленный репозиторий, рабочее дерево.

## Причина

Частые причины: команда запущена не из репозитория, есть незакоммиченные изменения, локальная ветка отстала от удаленной, один и тот же файл изменен в разных местах.

## Решение

```bash
git status
git diff
```

Дальше действовать по ситуации:

- если команда запущена не там — перейти в корень репозитория;
- если есть локальные изменения — сначала понять, какие файлы изменены;
- если конфликт — вручную выбрать итоговое содержимое и только потом коммитить;
- если ветка отстала — обновляться после проверки локальных изменений.

## Как не повторить

- Перед перемещениями и массовыми правками смотреть `git status`.
- Не смешивать в одном коммите заметки, код и временные файлы.
- Не добавлять в Git кэши, IDE-файлы и сборочные артефакты.
