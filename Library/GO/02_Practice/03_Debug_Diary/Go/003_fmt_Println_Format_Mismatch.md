# Автопроверка прошла, но формат вывода отличается от примера

## Ситуация

Код может пройти критерии курса, потому что содержит нужные вызовы и значения, но вывод визуально отличается от примера.

## Пример

```go
fmt.Println("Привет:", name)
```

Вывод:

```text
Привет: Гоша
```

А пример задания мог ожидать:

```text
Привет, Гоша
```

## Причина

`fmt.Println` автоматически расставляет пробелы между аргументами, но не управляет пунктуацией внутри строк.

## Решение

Для простого вывода можно поправить строковый литерал:

```go
fmt.Println("Привет,", name)
```

Для точного формата лучше использовать `fmt.Printf`:

```go
fmt.Printf("Привет, %s! Сейчас %s\n", name, formatted)
```

## Вывод

Если формат вывода важен, нужно явно управлять пунктуацией и переносом строки.
