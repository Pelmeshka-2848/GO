# Нельзя преобразовывать символ в цифру без проверки диапазона

## Ситуация

При реализации алгоритма Луна нужно получить числовое значение символа:

```go
digit := int(ch - '0')
```

## Ошибка

Если не проверить, что `ch` находится в диапазоне от `'0'` до `'9'`, нецифровой символ тоже будет преобразован в число.

Например:

```go
digit := int('a' - '0')
```

Это даст некорректное значение для алгоритма.

## Правильный вариант

```go
if ch < '0' || ch > '9' {
	return false
}

digit := int(ch - '0')
```

## Вывод

Перед преобразованием `byte` в цифру нужно проверить, что байт действительно является ASCII-цифрой.
