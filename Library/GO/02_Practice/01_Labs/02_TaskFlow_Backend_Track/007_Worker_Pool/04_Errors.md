# 04. Errors — Worker Pool

## Ошибки

Фактические ошибки пока не зафиксированы. Ниже список типовых проблем, которые стоит проверить во время выполнения.

| Ошибка | Причина | Решение | Связанная заметка |
|---|---|---|---|
| Deadlock | Канал не закрыт или goroutine ждёт данные, которые больше не придут | Явно определить владельца закрытия канала | [[01_Theory/07_Concurrency/Deadlock]] |
| Потеря результатов | Main завершается раньше workers | Дождаться завершения через `WaitGroup` или другой механизм синхронизации | [[01_Theory/07_Concurrency/WaitGroup]] |
| Data race | Несколько workers пишут в общую структуру без защиты | Использовать channel для результатов или mutex | [[02_Practice/03_Debug_Diary/Data_Race]] |

## Формат фиксации ошибки
