# 03. Code — Worker Pool

## Ключевой код

Код добавляется после самостоятельного выполнения лабораторной.

В разборе нужно показать:

- где создается очередь задач;
- где запускаются workers;
- как workers читают задачи из канала;
- как собираются результаты;
- как программа понимает, что все workers завершились.

## Файлы проекта

- `main.go` — запуск сценария и сборка pipeline.
- `worker.go` — логика worker и обработки задачи, если код будет вынесен из `main.go`.
- `README.md` — команды запуска и пример ожидаемого результата.
