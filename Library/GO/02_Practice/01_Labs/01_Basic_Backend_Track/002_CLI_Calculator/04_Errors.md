# Lab 002 - CLI Calculator — Errors

## Возможные ошибки

### Неверное количество аргументов

Причина: программа ожидает операцию и два числа, но получает меньше или больше значений.

Решение: добавить проверку количества аргументов до вычислений.

### Ошибка преобразования строки в число

Причина: пользователь передал нечисловое значение.

Решение: проверять ошибку после преобразования и выводить понятное сообщение.

### Деление на ноль

Причина: второй аргумент при операции деления равен нулю.

Решение: обработать этот случай до выполнения деления.

## Фактические ошибки

Заполнить после выполнения лабораторной.
