# Lab 002 - CLI Calculator — Task

## Цель

Закрепить базовую структуру CLI-приложения на Go: чтение аргументов, выбор операции, преобразование строк в числа и обработку ошибок.

## Задание

Создать учебный CLI-калькулятор, который принимает операцию и два числовых аргумента из командной строки.

Минимальные операции:

- сложение;
- вычитание;
- умножение;
- деление.

## Ограничения

- Не использовать сторонние библиотеки.
- Не писать интерактивный ввод, если основной сценарий строится на аргументах командной строки.
- Не игнорировать ошибки преобразования строк в числа.
- Для деления отдельно обработать деление на ноль.

## Критерии готовности

- [ ] Программа запускается через `go run`.
- [ ] Поддерживаются четыре базовые операции.
- [ ] Некорректные аргументы приводят к понятному сообщению об ошибке.
- [ ] Деление на ноль обработано отдельно.
- [ ] Команды проверки записаны в `02_Worklog.md`.
