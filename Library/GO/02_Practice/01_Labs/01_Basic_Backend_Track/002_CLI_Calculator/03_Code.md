# Lab 002 - CLI Calculator — Code

## Назначение

Файл фиксирует ключевые фрагменты уже написанного решения и объяснение логики CLI-калькулятора.

## Файлы проекта

| Файл | Назначение |
|---|---|
| `main.go` | точка входа, чтение аргументов, вызов логики расчета |
| `go.mod` | описание Go-модуля |

## Ключевые элементы

- чтение аргументов командной строки;
- проверка количества аргументов;
- преобразование строк в числа;
- выбор операции;
- обработка ошибок.
