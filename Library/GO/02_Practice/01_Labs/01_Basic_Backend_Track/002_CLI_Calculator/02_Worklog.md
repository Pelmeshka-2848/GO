# Lab 002 - CLI Calculator — Worklog

## План выполнения

1. Создать отдельную папку проекта.
2. Инициализировать Go-модуль.
3. Определить формат запуска CLI-команды.
4. Разобрать аргументы командной строки.
5. Преобразовать строковые аргументы в числа.
6. Добавить выбор операции.
7. Проверить успешные и ошибочные сценарии.

## Команды

```bash
go mod init example/cli-calculator
go run .
go run . add 2 3
go run . div 10 0
```

## Проверки

- Успешный расчет.
- Неизвестная операция.
- Недостаточно аргументов.
- Нечисловой аргумент.
- Деление на ноль.

- Ошибки перенести в `04_Errors.md`.
