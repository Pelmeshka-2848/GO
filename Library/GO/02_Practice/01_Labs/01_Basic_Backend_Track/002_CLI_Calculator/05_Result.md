# Lab 002 - CLI Calculator — Result

## Проверка

- [ ] Приложение запускается из терминала.
- [ ] Пользователь может выбрать арифметическую операцию.
- [ ] Ввод чисел обрабатывается ожидаемо.
- [ ] Ошибочный ввод не ломает сценарий без объяснения.

## Итог

В ходе выполнения работы был разобран базовый CLI-сценарий: чтение ввода, выбор операции, выполнение простого вычисления и вывод результата в терминал.

## Что стало понятнее

- CLI-программа должна явно работать с вводом и выводом.
- Проверка пользовательского ввода важна даже в маленькой учебной задаче.
- Простую задачу легче поддерживать, если отделять ввод, вычисление и вывод.
