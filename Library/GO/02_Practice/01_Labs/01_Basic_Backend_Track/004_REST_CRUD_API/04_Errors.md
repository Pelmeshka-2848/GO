# Lab 004 - REST CRUD API — Errors

| Ошибка | Причина | Решение | Связанная заметка |
|---|---|---|---|
| Неверный HTTP-метод для операции | CRUD-действие не сопоставлено с семантикой HTTP | Использовать `GET`, `POST`, `PUT`/`PATCH`, `DELETE` по назначению | [[01_Theory/05_HTTP_Backend/02_API_Practices/REST API]] |
| Пустой или невалидный JSON | Тело запроса не проверяется перед обработкой | Добавить валидацию входных данных и понятный ответ об ошибке | [[01_Theory/05_HTTP_Backend/02_API_Practices/Request Validation]] |
| Неподходящий HTTP-статус | Ошибка возвращается как обычный успешный ответ | Разделить успешные и ошибочные сценарии по статусам | [[01_Theory/05_HTTP_Backend/01_HTTP_Fundamentals/HTTP_Status_Codes]] |
