# Lab 005 - PostgreSQL CRUD — Task

## Задание

Расширить CRUD-сценарий до работы с PostgreSQL. Лабораторная должна закрепить подключение к базе, выполнение SQL-запросов и отделение HTTP-слоя от слоя хранения.

Минимальный набор действий:

- подготовить таблицу для одной сущности;
- настроить строку подключения;
- реализовать операции создания, чтения, обновления и удаления на уровне SQL;
- зафиксировать команды проверки и типовые ошибки подключения.

## Ограничения

- Не хранить пароль и строку подключения прямо в заметке как реальный секрет.
- SQL-запросы должны быть параметризованными.
- Ошибки базы должны обрабатываться отдельно от ошибок HTTP-запроса.
- Docker не обязателен, если PostgreSQL установлен локально.

## Критерии готовности

- [ ] Описана таблица и поля сущности.
- [ ] Зафиксирована схема подключения к PostgreSQL.
- [ ] Описаны CRUD-операции на уровне SQL.
- [ ] Отдельно записаны возможные ошибки подключения и миграций.
