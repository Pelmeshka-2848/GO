# Lab 005 - PostgreSQL CRUD — Worklog

## Ход работы

### Шаг 1. Подготовка базы

```bash
createdb <db-name>
psql <db-name>
```

### Шаг 2. Подключение приложения

- определить DSN или переменные окружения;
- открыть соединение с базой;
- проверить подключение через `Ping`;
- закрывать соединение при завершении приложения.

### Шаг 3. CRUD через SQL

- `INSERT` для создания записи;
- `SELECT` для чтения списка и одной записи;
- `UPDATE` для изменения записи;
- `DELETE` для удаления записи.

## Что получилось

- Лабораторная связывает REST CRUD с постоянным хранением данных.
- Отдельно выделены база, SQL-запросы и HTTP-слой.
