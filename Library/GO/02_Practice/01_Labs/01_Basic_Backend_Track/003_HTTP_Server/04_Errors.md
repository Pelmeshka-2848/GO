# Lab 003 - HTTP Server — Errors

## Возможные ошибки

### Порт уже занят

Причина: на выбранном порту уже запущен другой процесс.

Решение: остановить старый процесс или выбрать другой порт.

### Handler не вызывается

Причина: путь запроса не совпадает с зарегистрированным маршрутом.

Решение: проверить URL, метод запроса и регистрацию маршрута.

### Сервер завершился сразу после запуска

Причина: ошибка запуска не была обработана или программа дошла до конца без блокирующего вызова сервера.

Решение: явно запускать HTTP-сервер и проверять возвращаемую ошибку.

## Фактические ошибки

Заполнить после выполнения лабораторной.
