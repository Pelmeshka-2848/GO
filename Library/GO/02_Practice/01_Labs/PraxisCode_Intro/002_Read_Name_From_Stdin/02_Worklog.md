# 02 — Ход выполнения

## Выполненные шаги

1. Был подключён пакет `bufio` для работы со сканером.
2. Был подключён пакет `os` для доступа к стандартному вводу `os.Stdin`.
3. Был создан сканер через `bufio.NewScanner(os.Stdin)`.
4. Метод `scanner.Scan()` считал одну строку из входного потока.
5. Метод `scanner.Text()` вернул считанный текст.
6. Полученное значение было использовано в приветствии.

## Итог выполнения

Программа считывает имя из стандартного ввода и выводит приветствие.
