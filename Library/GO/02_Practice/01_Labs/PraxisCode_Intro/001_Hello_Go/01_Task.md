# 01 — Задание

## Тема

Привет, Go

## Цель

Освоить минимальную структуру исполняемой программы на Go и вывести значение переменной в терминал.

## Формулировка задания


В функции `main` была дана переменная `name`. Нужно было добавить вызов `fmt.Println`, чтобы вывести приветствие с использованием значения переменной `name`.

Важное условие: имя нельзя было вписывать вручную в строковый литерал. В вывод должен был попасть результат использования переменной.


## Критерии проверки

- В `func main()` вызывается `fmt.Println`.
- Один из аргументов вызова — переменная `name`, а не строковый литерал с именем.
- Аргументы `fmt.Println` содержат строку `Привет`.
