# 03 — Код

## Итоговое решение

```go
package main

import "fmt"

func main() {
	name := "Гоша"
	fmt.Println("Привет,", name)
}
```

## Примечание

Код соответствует критериям проверки курса. Где требовалось, решение дополнено более чистым стилем для дальнейшего использования в базе знаний.
