# 02 — Ход выполнения

## Выполненные шаги

1. Был объявлен пакет `main`.
2. Был подключён стандартный пакет `fmt`.
3. В функции `main` была создана переменная `name`.
4. Для вывода строки и значения переменной был использован вызов `fmt.Println`.

## Итог выполнения

Программа выводит приветствие с именем из переменной `name`.
