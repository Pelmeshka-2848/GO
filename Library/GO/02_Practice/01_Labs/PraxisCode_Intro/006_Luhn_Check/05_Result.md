# 05 — Результат

## Статус

Проверка курса пройдена.

## Выполненные критерии

- Функция `LuhnCheck(cardNumber string) bool` объявлена.
- Пустая строка и строки с нецифровыми символами возвращают `false`.
- Цифры перебираются справа налево.
- Каждая вторая цифра удваивается, и при результате больше 9 вычитается 9.
- Итоговая сумма проверяется на делимость на 10.

## Итог

Функция проверяет номер карты по контрольной сумме алгоритма Луна.
