# 02 — Ход выполнения

## Выполненные шаги

1. Была объявлена функция `LuhnCheck` с возвратом `bool`.
2. Была добавлена проверка пустой строки.
3. Был создан аккумулятор суммы `sum`.
4. Флаг `doubleNext` был использован для удвоения каждой второй цифры.
5. Строка перебиралась справа налево через индексный цикл.
6. Каждый байт проверялся на диапазон `'0'` — `'9'`.
7. ASCII-цифра преобразовывалась в число через `int(ch - '0')`.
8. После удвоения значения больше 9 корректировались через `digit -= 9`.
9. Финальная проверка выполнялась через `sum%10 == 0`.

## Итог выполнения

Функция проверяет номер карты по контрольной сумме алгоритма Луна.
