# 04 — Ошибки

## Фактические ошибки

Критических ошибок не было. В исходном решении переменная называлась `time_now`; для Go предпочтительнее `now` или `timeNow`. Также исходный вывод не содержал фразу `Сейчас`, поэтому был улучшен формат вывода.
