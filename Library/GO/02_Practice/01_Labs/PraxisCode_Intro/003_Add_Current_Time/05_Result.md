# 05 — Результат

## Статус

Проверка курса пройдена.

## Выполненные критерии

- В `func main()` вызывается `time.Now()`.
- На результате `time.Now()` вызывается метод `Format`.
- Аргумент `Format` — строка `15:04`.
- Вывод содержит имя из `scanner.Text()` и отформатированное время.

## Итог

Программа считывает имя и выводит приветствие с текущим временем в формате `HH:MM`.

## Вывод по лабораторной

В ходе выполнения лабораторной работы была закреплена тема: **Добавь текущее время**. Полученный код можно использовать как основу для следующих задач курса PraxisCode Intro.
