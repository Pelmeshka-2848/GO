# 01 — Задание

## Тема

Добавь текущее время

## Цель

Научиться получать текущее время в Go с помощью `time.Now()` и форматировать его через метод `Format`.

## Формулировка задания


Нужно было изменить программу из предыдущей лабораторной так, чтобы она считывала имя из `stdin`, получала текущее время, форматировала его в виде `часы:минуты` и выводила приветствие с именем и временем.


## Критерии проверки

- В `func main()` вызывается `time.Now()`.
- На результате `time.Now()` вызывается метод `Format`.
- Аргумент `Format` — строка `15:04`.
- Вывод содержит имя из `scanner.Text()` и отформатированное время.
