# 02 — Ход выполнения

## Выполненные шаги

1. Было сохранено чтение имени через `bufio.Scanner`.
2. Был подключён пакет `time`.
3. Текущее время было получено через `time.Now()`.
4. Время было отформатировано через `Format("15:04")`.
5. Для точного управления форматом вывода был использован `fmt.Printf`.

## Итог выполнения

Программа считывает имя и выводит приветствие с текущим временем в формате `HH:MM`.
